# SwasthyaSetu Frontend

React + Vite frontend integrated with the SwasthyaSetu backend.

## Run
```bash
npm install
cp .env.example .env
npm run dev
```

Default API base URL:
- `VITE_API_BASE_URL=http://localhost:8080`

## Login (seeded accounts)
Use the role cards; each maps to backend seeded credentials from `backend/prisma/seed.ts`.
